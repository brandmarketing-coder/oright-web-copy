## GDP → GEP（Paradigm Shift）

| 欄位 | 文案 |
| --- | --- |
| 徽章 | Paradigm Shift |
| 主標 H2 | 重新定義價值：從 GDP 到 GEP |
| 副標 | 歐萊德將工業角度提升至生態層次：GDP 衡量產出，卻讓自然成為沉默成本；GEP（Gross Ecosystem Product）讓每一片森林、每一條河流，都有被計算的理由。 |

**左欄：GDP 舊思維**

-   石化原料廉價、穩定，是美妝業最省事的選擇
-   廢棄農產品與大地資源遭忽視，淪為無用廢料
-   碳排放是「外部成本」，不進損益表

**右欄：GEP 新價值 🌱**

-   USDA Biobased 100%：讓廢棄農產品萃取原料賦予新生，取代石化來源
-   100% PCR 再生塑料：從設計起就為回收而生，減碳 83.7%
-   ISO 14068-1 碳中和認證：讓每噸排放都有所依據、有所抵消
