## GDP → GEP（Paradigm Shift）

| 欄位 | 文案 |
|---|---|
| 徽章 | Paradigm Shift |
| 主標 H2 | 從計算金錢，到計算地球 |
| 副標 | GDP 衡量產出，卻讓自然生態成為沉默的成本。GEP（Gross Ecosystem Product）讓每一片森林與河流，都有被看見的理由。 |

**左欄：GDP 的盲點**
- 將自然資源視為免費原料
- 以短期產值掩蓋長期環境負債
- 生態系統崩潰無法列入損益表

**右欄：GEP 新框架 🌱**
- 量化自然服務的真實經濟價值
- 讓企業決策與生態健康掛鉤
- 復育生態即是創造長期可持續資產
