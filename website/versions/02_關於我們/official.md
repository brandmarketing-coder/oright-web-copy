## GDP → GEP（Paradigm Shift）

| 欄位 | 文案 |
| --- | --- |
| 徽章 | Paradigm Shift |
| 主標 H2 | 重新定義價值：從 GDP 到 GEP |
| 副標 | 歐萊德將工業角度帶向更深層面：GDP 衡量產出，卻讓自然生態成為沉默成本；GEP（Gross Ecosystem Product）讓每一片森林、每一條河流，都有被計算的理由。 |

**左欄：GDP 舊思維**

-   石化原料廉價、穩定，是最省事的美妝業多年選擇
-   廢棄農副物與大地資源遭忽視，淪為無用廢棄
-   碳排放是「外部成本」，不進損益表

**右欄：GEP 新價值 🌱**

-   USDA Biobased 100%，讓廢棄農副物取得新生，取代石化來源
-   100% PCR 再生包裝，從設計起就為回收而生，減碳 83.7%
-   ISO 14068-1 碳中和認證，讓每次排放都有依據、有流向
