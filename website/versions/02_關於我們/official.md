## GDP → GEP（Paradigm Shift）
> 內容形式：【兩欄比較】

| 欄位 | 文案 |
| --- | --- |
| 徽章 | Paradigm Shift |
| 主標 H2 | 重新定義價值：從 GDP 到 GEP |
| 副標 | GDP 衡量產出，卻讓自然生態成為沉默的成本。GEP（Gross Ecosystem Product）讓每一片森林與河流，都有被看見的理由。 |

**左欄：GDP 舊思維**

-   石化原料廉價、穩定，是最省事的選擇
-   塑膠包裝成本低，回收責任留給下游
-   碳排放是「外部成本」，不進損益表

**右欄：GEP 新價值 🌱**

-   USDA Biobased 100% 農業副產物取代石化原料
-   PCR 再生包裝從設計起就為回收而生
-   ISO 14068-1 碳中和認證讓排放有所依據
