## GDP → GEP（Paradigm Shift）

| 欄位 | 文案 |
|---|---|
| 徽章 | Paradigm Shift |
| 主標 H2 | 美妝業的隱形帳單：自然，不該是免費原料 |
| 副標 | GDP 讓美妝產業把石化資源視為理所當然。GEP（Gross Ecosystem Product）讓我們看見：每一瓶洗髮精，都與生態健康有關。 |

**左欄：產業的舊邏輯**
- 石化原料廉價、穩定，是最省事的選擇
- 塑膠包裝成本低，回收責任留給下游
- 碳排放是「外部成本」，不進損益表

**右欄：O'right 的回應 🌱**
- USDA Biobased 100% 農業副產物取代石化原料
- PCR 再生包裝從設計起就為回收而生
- ISO 14068-1 碳中和認證讓排放有所依據
